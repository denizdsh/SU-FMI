# Задача 3: Анализ на студенти

## Условие:
Създайте функция `analyzeStudents`, която анализира масив от студенти.
(Винаги има масив за входни данни)

### Входни данни:
Масив от обекти със структура:
```javascript
{
    name: "string",
    grades: [numbers], // масив от оценки
    birthYear: number
}
```

### Изходни данни:
Функцията трябва да връща обект със следните property-та:

- **`topStudent`** - студентът с най-висок среден успех (целия обект) или `"unknown"` ако няма студенти или ако не може да се изчисли
- **`averageGrade`** - общата средна оценка от всички оценки на всички студенти (закръглена до 2 знака) или `"unknown"` ако не може да се изчисли
- **`passedCount`** - брой студенти със среден успех >= 3.00 или `0` ако не може да се изчисли
- **`ageStats`** - обект с възрастова статистика:
  - `youngest` - възраст на най-младия студент или `"unknown"`
  - `oldest` - възраст на най-възрастния студент или `"unknown"`

### Тестови случаи за оценка:
```javascript
const students = [
    { name: "Иван", grades: [5, 4, 6], birthYear: 2000 },
    { name: "Мария", grades: [6, 6, 5], birthYear: 1999 },
    { name: "Петър", grades: [3, 2, 4], birthYear: 2001 }
];

console.log(analyzeStudents(students));
```
